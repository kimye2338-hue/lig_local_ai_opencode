### 필수 2 — owner 추출 (리뷰어가 5개 케이스로 실측 통과시킨 코드)
`_meeting_actions` 안의 owner 블록을 아래 헬퍼로 교체. **후행(담당: 이름) 우선 →
토큰이 날짜형이면 거부 → 선행(이름 담당) fallback → 그래도 없으면 "확인 필요"** 순서다:

```python
_OWNER_DATE_LIKE = re.compile(r"^\d|^\D*\d|[월일주후시]$")

def _extract_owner(sentence: str) -> str:
    m = re.search(r"담당[:：]?\s*([A-Za-z0-9_가-힣]+)", sentence)   # 담당: <이름>
    if m and not _OWNER_DATE_LIKE.search(m.group(1)):
        return m.group(1)
    m = re.search(r"([가-힣]{2,4})\s*담당", sentence)               # <이름> 담당
    if m:
        return m.group(1)
    return "확인 필요"
```

리뷰어 실측 결과(전부 통과): `김대리 담당: 7월 10일까지…`→`김대리`,
`도면 수정 담당: 김대리…`→`김대리`, `7월 10일까지 도면…`→`확인 필요`,
`담당: 이과장, 케이블…`→`이과장`, `브라켓 두께 3mm 늘리기로 합의했다`→`확인 필요`.

todo 정리는 owner로 확정된 토큰과 due 구절을 함께 제거:

```python
owner = _extract_owner(sentence)
todo = sentence
if owner != "확인 필요":
    todo = re.sub(rf"{re.escape(owner)}\s*담당[:：]?", "", todo)
    todo = re.sub(r"담당[:：]?\s*[A-Za-z0-9_가-힣]+", "", todo)
if due != "확인 필요":
    todo = todo.replace(due, "")
todo = re.sub(r"\s+", " ", todo).strip(" ,.")
```

bench 정확성 check:

```python
check("meeting action owner is a person or 확인 필요, never a date fragment",
      ("| 김대리 |" in minutes) and ("| 7월 |" not in minutes), minutes[:1200])
```
