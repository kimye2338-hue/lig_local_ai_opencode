- **완전 제거를 원할 경우 옵션**: (a) 소유자가 GitHub에서 닫힌 PR #1/#2/#4 자체를
  삭제, 또는 (b) 서버 gc가 미참조 pull ref를 회수하도록 대기. 둘 다 플랫폼 측 조치라
