## 목표
회사와 동일 조건(인터넷 0)에서 번들 설치가 끝까지 되는지 리허설.

## 절차
1. build_bundle.py로 zip 생성 → 다른 폴더(가상의 "회사 PC" 루트)에 복사.
2. **사람**: 네트워크 어댑터 비활성화.
3. zip 해제 → setup.bat 실행 → doctor 전 섹션 결과 기록 (스크린샷/텍스트).
4. `work --task "..." --mode mock` 1회 + briefing.bat 1회 실행 확인.
5. **사람**: 네트워크 복구. 결과를 `plan/reports/P17-04-r1.md`에 기록
   (실패 항목은 원인과 함께 — 실패가 이 작업의 정상 산출물일 수 있음).
