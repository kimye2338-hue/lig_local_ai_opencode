# CNC (컴퓨터수치제어)

관련: [[기계공작법]], [[solidworks]], [[치구설계]].
> ⚠️ G코드는 FANUC 기준. Haas/Siemens(Sinumerik)/Mitsubishi는 코드 배정 다름 — 기계 컨트롤러 매뉴얼 확인.

## 1. NC와 CNC
NC(1952 MIT, 천공테이프·하드와이어드)→CNC(1970s 마이크로프로세서, 프로그램 메모리 저장·SW 해석).
구성: 컨트롤러(FANUC/Siemens/Mitsubishi/Haas), 서보모터(폐루프 위치/속도 피드백), **볼스크류**(회전→직선, 백래시 작음), **엔코더**(로터리=간접/리니어스케일=직접). 개루프(스텝·피드백 없음, 저가) vs **폐루프**(엔코더 보정, 산업용 표준).

## 2. 좌표계
- **기계원점**: 제작사 고정 기준(전원 시 원점복귀 G28로 재획득). 변경 불가.
- **공작물원점 G54~G59**: 워크 오프셋(에지파인더/프로브로 측정→오프셋 테이블 입력). 부족 시 G54.1 P1~P48.
- **G90 절대**(활성 워크원점 기준, 실무 표준) vs **G91 증분**(직전 위치 기준, 반복패턴).
- **평면 G17(XY 밀링 표준)/G18(XZ 선반)/G19(YZ)** — 원호보간·공구경보정 평면.

## 3. G코드 (FANUC 준비기능)

### 3.1 이동
- **G00 급속이송**: 비절삭 최대속(비선형 보간). 안전높이 Z 확보 후 XY.
- **G01 직선보간**: F 이송속도로 직선 절삭(윤곽·포켓·평면 대부분).
- **G02(CW)/G03(CCW) 원호보간**: I,J,K(시작점→중심 증분) 또는 R(반경, 180°초과는 R음수). 예 `G02 X70 Y20 I20 J0 F100`.
- **G04 드웰**: `G04 P1000`(1초, P=ms 또는 X=초).

### 3.2 단위·원점
G20 인치/G21 mm(최상단 1회, 국내 G21 표준). **G28** 자동 원점복귀(경유점→기계원점, 전원 후 1회 필수·ATC 전).

### 3.3 공구경보정(CRC) G40/41/42
프로그램은 최종 윤곽 좌표, 컨트롤러가 공구반경만큼 오프셋. **G41 좌측(상향절삭)/G42 우측(하향절삭)/G40 취소**. 리드인/아웃(공작물 밖 진입) 필요, 보정량보다 작은 내측코너는 간섭 알람. `G41 D01 ...`(D=반경 오프셋).

### 3.4 공구길이보정 G43
Z 지령에 공구별 길이 오프셋 가산. `G43 H01 Z50`, G49 취소.

### 3.5 고정사이클 G80~G89
급속접근→절삭이송→동작→급속후퇴 압축.
| 코드 | 용도 |
|---|---|
| G73 | 고속 심공드릴(칩브레이크) |
| G74 | 역태핑 |
| G76 | 정밀보링(바닥 shift 후퇴) |
| G80 | 취소 |
| G81 | 단순 드릴 |
| G82 | 카운터보어(바닥 드웰) |
| G83 | 심공드릴(완전 후퇴 칩배출) |
| G84 | 태핑(리지드 vs 플로팅) |
| G85/86/89 | 보링(정삭/주축정지/드웰) |
어드레스: X,Y 위치·Z 바닥·R 전환높이·Q 펙·P 드웰·F 이송. G98 초기점/G99 R점 복귀. 예 `G90 G99 G81 X20 Y20 Z-15 R5 F80` 후 좌표만 반복(모달), G80.

### 3.6 주속일정 G96/G97
선반. **G96 CSS**(표면속도 m/min 일정, RPM 자동변화 — 단차·페이싱 필수). **G97 RPM 고정**(센터드릴 등).

## 4. M코드(보조기능)
| 코드 | 의미 |
|---|---|
| M00 | 무조건 정지(측정 후 재개) |
| M01 | 옵셔널 정지 |
| M02 | 종료(리와인드 없음) |
| M03/04/05 | 주축 정회전/역회전/정지 |
| M06 | 공구교환(ATC, T코드와 짝) |
| M08/09 | 절삭유 ON/OFF |
| M30 | 종료+리와인드 |
예: `S1200 M03` `M08` … `M09 M05 M30`.

## 5. 밀링 프로그래밍
- **포켓**: 헬리컬/램프 Z진입(수직 플런지 지양)→나선/지그재그 러핑→벽 정삭(G41/42)→바닥 정삭.
- **윤곽**: 공구경보정으로 도면치수 그대로, 진입/이탈은 접선 원호/램프(언더컷 방지).
- **헬리컬 보간**: G02/03+Z 동시 → 나선. 엔드밀 하나로 대구경 구멍·나사. `G02 X50 Y0 Z-2 I-25 J0 F100`.
- **상향절삭**(회전≠이송, 칩 0→최대, 백래시 둔감) vs **하향절삭**(회전=이송, 칩 최대→0, 표면·수명 우수, 현대 CNC 표준).

## 6. 선삭 프로그래밍
좌표 다름(X=직경, Z=주축, G18 평면).
- **G71 황삭**: 프로파일(P~Q 블록)을 절삭깊이 U만큼 여러 패스. `G71 U2 R1` `G71 P100 Q200 U0.5 W0.2 F0.2`.
- **G70 정삭**: P~Q 프로파일 정삭여유 없이 마무리. `G70 P100 Q200`.
- **G76 나사절삭**: 여러 패스, 두 줄 형식 `G76 P(m)(r)(a) Q(dmin) R(d)` / `G76 X Z R P Q F(피치)`. 주축-이송 동기(엔코더).
- **공구노즈R 보정(TNRC)**: 인서트 노즈반경(R0.2~0.8) 보정, G41/42+가상절삭점 방향(0~9).

## 7. 5축·다축
3직선축+회전 2축(A/B/C). **3+2 위치결정**(회전축 고정 후 3축, 리클램핑↓) vs **동시 5축**(절삭 중 자세 변화, 블레이드·임펠러, RTCP 필수, CAM 필수). 테이블/헤드/혼합형. 특이점(singularity) 회피 CAM 검토.

## 8. CAM 워크플로우
① CAD 모델·스톡([[solidworks]]) ② 툴패스 생성(전략·공구·조건→CL Data) ③ 시뮬(과다절삭·간섭·잔량) ④ **포스트프로세서**(CL Data→기계별 G/M코드 방언, 기계마다 커스텀) ⑤ 드라이런/에어컷→실가공.
결정사항: 공구목록·홀더간섭, 절삭조건([[기계공작법]]), 가공순서, 워크오프셋 매핑(CAM 원점↔G54).

## 9. 공구·홀더·절삭조건
재종(HSS/초경/세라믹/CBN/PCD)·코팅·홀더·절삭속도/이송/절입 상세는 [[기계공작법]]. 프로그램엔 S·F·CAM 절입 파라미터로 반영.

## 10. 좌표측정·셋업
- **에지파인더**: 수동 회전, 모서리 접촉으로 X/Y 위치(G54 계산).
- **프로브(터치)**: 전자 접촉센서, 자동 좌표 기록·세팅 매크로·온머신 검사.
- **공구 프리세터**: 외부서 공구 길이/직경 사전측정→오프셋 입력.

## 11. 알람·충돌 방지
알람: 오버트래블(원점복귀 미실행·좌표 오타), 서보(follow error), 공구경보정 간섭(코너<공구반경), 문법오류.
충돌 예방: CAM 시뮬(홀더·클램프 포함), 신규는 드라이런/싱글블록+오버라이드↓, **워크오프셋+공구오프셋 재확인(공구교환 후 오프셋 오적용이 최다 원인)**, Z 안전높이 충분히.
