# 엔지니어링 데이터처리

## ⚠️ 핵심 주의 (항상)
- **샘플링 정리(Nyquist)**: 왜곡 없이 복원하려면 `fs ≥ 2×fmax`. fs/2가 나이퀴스트 주파수. 초과 성분은
  **에일리어싱**(가짜 저주파, 예: 10kHz 샘플링의 12kHz→2kHz 오인). FFT/PSD 전 계측 fs·앤티에일리어싱
  필터 여부 확인. 불확실하면 사용자에게 fs·필터 먼저 확인.
- **창함수·단위 없는 FFT/PSD 결과는 불신**: 창(Hann 흔함)·오버랩(기본 50%)·정규화(one/two-sided,
  density vs spectrum)에 따라 절대값 크게 달라짐(scipy.signal.welch 기준).
- **이상치 기계적 삭제 금지**: 노이즈인지 실제 물리현상(과도·충격)인지 원인 파악 우선. 삭제/보간 시 원본
  백업·처리이력 남김.
- **재현성**: 처리 스크립트+파라미터(fs·창·필터 차수/차단)를 결과와 함께 보관.

## 데이터 정제·기초통계
- 결측: `pandas isna()/dropna()/interpolate()`(물리 타당 구간만 보간). 이상치: IQR 또는 ±3σ로 flag 후 검토(자동삭제 아님).
- **평균·표준편차·RMS는 다른 개념**. 평균 0인 순수 AC는 RMS=std, DC 오프셋 있으면 `RMS²=mean²+std²`.
- Python: `df.describe()`, RMS는 `np.sqrt(np.mean(x**2))`.

## 신호처리 (샘플링·필터·FFT·PSD)
- 샘플링: fs 항상 확인, `fs ≥ 2×fmax`(여유로 5~10×fmax 권장 — ⚠️관례치, 표준 인용 아님).
- **필터**(위상지연 없는 양방향):
  ```python
  from scipy.signal import butter, filtfilt
  b, a = butter(N=4, Wn=cutoff_hz/(fs/2), btype='low')  # Wn은 나이퀴스트 정규화 0~1
  y_f = filtfilt(b, a, y)
  ```
- **FFT**(실수신호는 rfft):
  ```python
  import numpy as np
  n=len(y); w=np.hanning(n)                 # 누설 억제
  yf=np.fft.rfft(y*w); freqs=np.fft.rfftfreq(n, d=1/fs)
  amp=np.abs(yf)/n*2                          # 창 이득 보정 별도 고려
  ```
- **PSD(Welch 권장, 노이즈 강건)**: 겹치는 구간 주기도 평균. 기본 창 Hann·오버랩 50%.
  ```python
  from scipy.signal import welch
  f, Pxx = welch(y, fs=fs, window='hann', nperseg=1024, noverlap=512)
  ```

## 진동데이터 특화 (Grms·PSD 적분 — [[진동해석]] 연계)
- **Grms** = PSD(g²/Hz) 곡선 아래 면적을 관심대역 적분 후 제곱근: `Grms = sqrt(∫ PSD(f) df)`.
- PSD는 로그-로그 표현이 흔해 **구간별 로그기울기 적분** 필요(단순 선형 사다리꼴은 저주파 급경사서 오차 큼).
- ⚠️ 시험 스펙 판정기준([[MIL-STD-810H]] 등)은 원문 별도 확인(본 노트 미인용).

## 시각화·리포트
- 시계열 x=시간(단위 명시), 이벤트 구간 표시. 스펙트럼: 진폭은 선형/dB, PSD는 로그-로그(진동/음향 관례).
- 그래프에 fs·창·필터 조건을 캡션/범례에 명기(재현성).
