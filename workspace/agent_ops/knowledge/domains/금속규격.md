# 금속규격 (Metal Standards & Designations)

> 도면·BOM·조달 문서의 강종·합금 표기를 읽고 규격 체계 간 관계를 파악하는 실무 레퍼런스.
> [[재료역학]]의 규격 체계 절을 실제 표기법으로 심화. 관련: [[기계공작법]](열처리), [[GD&T]](도면).
> ⚠️ 강종별 정확한 기계적 성질(Sy/Sut)은 싣지 않는다 — 규격 원문/재료증명서(MTR)/MMPDS 최신판 확인.

## 표기 읽는 법 요약
| 표기 | 체계 | 읽는 법 |
|---|---|---|
| `SM490A`/`SS400` | JIS/KS 구조용강 | SM=용접구조용·SS=일반구조용, 숫자=인장강도 하한(MPa)대, A/B/C=충격등급 |
| `4340` | AISI/SAE | 4x=Ni-Cr-Mo계, 3=Ni대, 40=탄소 0.40% |
| `304`/`316L`/`17-4PH` | AISI 스테인리스 | 3xx=오스테나이트·4xx=마르텐사이트/페라이트·PH=석출경화·L=저탄소 |
| `S17400`/`G43400` | UNS | S=스테인리스·G=탄소/합금강 + 5자리(17-4PH·4340) |
| `6061-T6`/`7075-T73` | 알루미늄 | 6=Mg-Si·7=Zn계, T6=용체화+인공시효, T73=SCC저항 과시효 |
| `Ti-6Al-4V`/`Grade 5` | ASTM Ti | 조성표기=Grade 5, α+β형 |
| `AMS 5643` | SAE AMS | 항공우주 재료 스펙(예 17-4PH 봉재) |
| `MIL-HDBK-5`/`MMPDS` | 구 군용/현 FAA | 항공 구조 설계허용치(A/B-basis). 5J는 2004 폐지→MMPDS 계승 |

## 1. 규격 체계 개요·상호관계
| 체계 | 기관 | 성격 | 용도 |
|---|---|---|---|
| KS | KATS(한국) | 국가표준(JIS 영향) | 국내 조달·도면 |
| JIS | JISC(일본) | 국가표준 | 국내 기계·조선 관행 다용 |
| ASTM | 민간(미국) | 시험법+형태별 스펙 | 범용 조달·배관/압력용기 |
| SAE/AISI | SAE/(구)AISI | 강종 화학조성 채번 | 자동차·기계 강종 표준 |
| AMS | SAE | 항공우주 재료 상세 스펙(조성·공정·시험·품질) | 항공/방산 조달 |
| EN/DIN | CEN/DIN | 유럽 통합·독일(EN 대체 중) | 유럽계 도면 |
| MIL-HDBK-5→MMPDS | 구 DoD→FAA | 구조 설계허용치(A/B-basis) | 구조해석 설계값 |
| UNS | ASTM+SAE | 합금 "성분 식별" 번호(완전 스펙 아님) | 규격 간 상호참조 키 |

**요점**: UNS는 규격 간 동일합금 상호참조 키(단독 구매스펙 불가 — 열처리·형태·품질·시험 미포함). MIL-HDBK-5J는 2004 폐지, MMPDS(FAA)가 계승·정기개정 — 구 도면에 "MIL-HDBK-5"라도 최신 설계값은 MMPDS 확인. AMS(제조·시험 방법)와 MMPDS(설계 허용강도)는 상호보완. KS↔JIS 유사하나 자동 동일시 위험(§7).

### 1.2 번호 읽기 원칙
- ASTM: 접두(A=철강·B=비철)+순번(A36·B209), 문자=대분류·숫자=등록순.
- AMS: 4자리 스펙번호(조성 인코딩 아님, 원문 확인).
- UNS: 접두문자+5자리(§3). EN: 재료번호(1.4301)+기호(X5CrNi18-10) 병기.

## 2. 강 표기 체계
### 2.1 AISI/SAE 4자리
**1째=주 합금계열, 2째=함량대(1≈1%), 끝 2자리=탄소(0.01%)**. 예 4340=Ni-Cr-Mo, Ni대 3, C 0.40%.
| 계열 | 원소 | 특징 |
|---|---|---|
| 10xx | 탄소강 | 범용 저가 |
| 11xx | 쾌삭강(S) | 자동선반 |
| 41xx | Cr-Mo | 관통경화·인성, 축·기어 |
| 43xx | Ni-Cr-Mo | 고강도·고인성(4340 벤치마크) |
| 46/48xx | Ni-Mo | 침탄 |
| 51xx | Cr | 스프링·베어링 |
| 52100 | 고탄소 Cr | 베어링 표준 |
| 61xx | Cr-V | 스프링·공구 |
| 86/87xx | Ni-Cr-Mo(저합금) | 침탄 기어·축 |
| 92xx | Si-Mn | 스프링 |
(화학조성·물성은 ASTM/SAE 원문·MTR 확인.)

### 2.2 스테인리스 3자리 + PH
| 계열 | 조직 | 특징 | 예 |
|---|---|---|---|
| 2xx | 오스테나이트(Mn-Ni) | 저Ni 대체·비자성 | 201, 202 |
| 3xx | 오스테나이트(Cr-Ni) | 다수·비자성·냉간가공만 강화 | 304, 316L |
| 4xx 페라이트 | 페라이트 | Cr단독·자성·경화 안 됨 | 409, 430 |
| 4xx 마르텐사이트 | 마르텐사이트 | 자성·열처리 경화 | 410, 420 |
| PH | 마르텐사이트 PH | 고용화 공급→가공 후 단일 시효 경화, 방산/항공 다용 | 17-4PH(AISI 630=S17400), 15-5PH, 17-7PH |
| 듀플렉스 | 오스테+페라이트 | 고강도+내식 균형 | 2205 |

### 2.3 공구강(AISI 문자)
W(수냉 탄소)·O(유냉 냉간)·A(공냉 냉간 중합금)·D(고탄소-고Cr 내마모)·S(내충격)·H(열간, H1x=Cr/H2x=W/H4x=Mo).

## 3. UNS 체계
접두 문자+5자리. ASTM·SAE 공동관리, 규격 간 동일합금 키(단독 스펙 아님).
| 접두 | 재료군 |
|---|---|
| Gxxxxx | 탄소/합금강(G43400=4340) |
| Sxxxxx | 스테인리스·PH·철계 초내열 |
| Axxxxx | 알루미늄 |
| Rxxxxx | 내화금속·특수(R5xxxx=Ti·R3xxxx=Co) |
| Nxxxxx | 니켈 |
| Cxxxxx | 동합금 |

## 4. 알루미늄 표기
### 4.1 전신재 4자리(1째=주 원소)
| 계열 | 원소 | 열처리 | 특징 |
|---|---|---|---|
| 1xxx | 순Al(99%+) | 비열처리 | 내식·전도·저강도 |
| 2xxx | Cu | 열처리 | 고강도 항공(2024), 내식 낮음(클래딩) |
| 3xxx | Mn | 비열처리 | 성형·내식 범용 |
| 5xxx | Mg | 비열처리 | 해양·내식·용접 양호 |
| 6xxx | Mg+Si | 열처리 | 압출·용접·내식 균형(6061) |
| 7xxx | Zn(+Cu/Mg) | 열처리 | 최고강도 항공(7075), SCC 관리 |
| 8xxx | 기타(Li·Fe) | 상이 | Al-Li 특수 |

### 4.2 조질(temper)
F(as fab)·O(소둔 최연질)·H(가공경화 1/3/5xxx, H1x/H2x/H3x)·T(열처리 T1~T10)·W(용체화 후 자연시효 중, 불안정).
**T 세부**(2/6/7xxx): T3(용체화+냉간+자연시효)·T4(용체화+자연시효)·T6(용체화+인공시효 피크강도)·T7(과시효 안정/내식)·T73(7075 SCC저항 특수 과시효).
예: 6061-T6(범용 구조·용접), 7075-T73(SCC저항 고강도 항공). 정확 강도는 합금-두께-조질별 상이(AMS/ASTM B209·MTR 확인).

## 5. 티타늄
ASTM Grade 분류(순Ti Grade 1~4, 합금 이후 번호). **Ti-6Al-4V(Al6·V4)=ASTM Grade 5**, α+β형, 항공/방산 대표. 조직: α(저온상)·β(고온상, 준안정 β존재)·α+β(절충, Ti-6Al-4V). 도면서 조성명 vs Grade vs AMS 스펙 어느 표기인지 확인.

|---|---|---|---|---|
| 저합금강 | 4340 | AISI 4340, AMS 6414 | G43400 | 고강도·고인성(축·랜딩기어) |
| SS 오스테나이트 | 304/316L | ASTM A240 | S30400/S31603 | 내식 배관·구조 |
| SS PH | 17-4PH | AISI 630, AMS 5643 | S17400 | 고강도+내식 |
| Al 범용 | 6061-T6 | ASTM B209/B221, AMS 4027 | A96061 | 범용 가공·구조 |
| Al 고강도 | 7075-T73 | ASTM B209, AMS 4045 | A97075 | SCC관리 항공구조 |

## 7. 상호참조 주의 — "등가 ≠ 동일"
국가규격 대응표(KS↔JIS↔ASTM↔EN)는 근사일 뿐 100% 동일 아님. 규격별로 다를 수 있는 것: 화학성분 상하한(불순물)·기계성질 시험(시편크기·방향 L/T)·충격시험 온도·열처리 상태·형태별 스펙. 방산/항공은 지정 규격을 임의 동등품 치환 금지 — 계약/도면 규격 그대로 조달 또는 정식 대체승인(MRB/재료대체). UNS 같아도 요구 시험/품질 수준은 다를 수 있음.

## 8. 재료증명서(MTR/Mill Cert)
규격=요구범위, MTR=로트 실측. 확인: 적용규격/개정판(ASTM A276-21)·히트/로트번호(추적성)·화학성분(상하한 이내)·기계성질(실측 vs 요구)·열처리상태(도면과 일치)·형상/치수(두께효과)·시험방법/인증기관(NADCAP)·원산지/제조사(방산 원산지 규정).

## 9. 열처리·조질 (강 vs 알루미늄)
| 구분 | 강 | 알루미늄 |
|---|---|---|
| 경화 원리 | 담금질+템퍼링 상변태 | 용체화+시효 석출(2/6/7xxx) 또는 가공경화(1/3/5xxx) |
| 표기 | 별도 상태문자(QT) 또는 스펙 내 지정 | 합금번호+조질기호(-T6/-H14) |
| 대표 | annealed·QT·normalized | O·H·T·W |
| 도면 예 | "AISI 4340, QT to Rc 40-44" | "7075-T73" |
세부 절차(침탄·질화·오스템퍼)·물성 변화는 [[기계공작법]].

## 10. 방산 조달 실무
- 도면/BOM에 규격+개정판+등급 정확 인용("스테인리스" 뭉뚱그림은 리스크).
- 형태별 스펙 분리(봉/판/단조/주조 각각 다른 ASTM/AMS) — 형태 함께 지정.
- 안전 중요부는 상용 ASTM보다 엄격한 AMS 지정 흔함.
- 설계허용치는 MMPDS 최신판 A/B-basis, 사용 edition 명기.
- 원산지 제한·NADCAP 인증 요건 견적 전 확인.
- 강도 인용은 "MMPDS Rev.XX, Table X 기준"으로 출처 판 명시(카탈로그 수치 그대로 인용 금지).
