# 구조해석 기초

## ⚠️ 핵심 개념·주의 (항상 확인)
- **응력(Stress) vs 변형률(Strain) vs 변위(Displacement) 혼동 금지.** 파괴/항복 판정은 응력 기준이며, 변위(처짐)만 작아도 국부 응력집중부는 항복·파괴할 수 있음.
- **안전율(FoS) 정의 확인**: FoS = 허용응력(항복 또는 극한) / 실제 발생응력. 여유율(MS) = FoS − 1 (표준마다 정의가 달라 사용 규격 정의를 반드시 확인).
- **항공우주 표준 안전율**: FAR 기준 극한 안전율(Ultimate FoS) **1.5**가 항공기 표준, 우주비행체는 **1.4** 사용 사례 다수, 착륙장치 등 1.25도. 값은 **프로젝트 규격/계약 요구가 최우선**, 이 수치는 참고치 [NASA/TM-2015].
- **경량화 압박**: 항공우주는 낮은 안전율을 쓰는 대신 하중 산정·물성·해석 정확도 검증이 훨씬 엄격. 안전율이 낮을수록 하중 케이스 누락·해석 오류의 대가가 큼 — 정적+동적+열+진동/충격 하중 스펙트럼을 모두 포함했는지 확인.
- **유도탄/방산 맥락**: 발사 축가속도, 로켓모터 연소진동, 공력가열, 취급/운송 충격, 사격/발사 충격 등 **다중 하중 조합** — 정적해석 하나로 대체 불가. 진동·충격 해석/시험 병행 필수.

## 하중·응력·변형
- 후크의 법칙(선형탄성). 항복점 이후는 비선형(소성) 별도 해석.
- 등가응력(von Mises)은 다축응력을 단일 스칼라로 환산해 연성 금속 항복 판단:
  σ_vm = √[ ½((σx−σy)²+(σy−σz)²+(σz−σx)²) + 3(τxy²+τyz²+τzx²) ]
  (2D 단순화: σ_vm = √(σ1²+σ2²−σ1σ2)) [Engineer's Edge].
- von Mises는 **연성(금속)**용. 취성(세라믹·주철 등)은 최대주응력 등 별도 파괴이론 — 재료 연성 먼저 확인(사내 재료 데이터시트 권장).

## 파괴 기준 (von Mises 등)
- 연성 금속: von Mises가 실무 표준. 항복강도(Sy) 대비 안전율.
- 판정: (1) σ_vm 확인 → (2) Sy 대비 여유율 → (3) 극한강도(Su) 대비 극한 안전율도 함께(항공우주는 항복+극한 이중 기준) [NASA/TM-2015].
- 반복하중(피로)이 있으면 정적 판정만으론 불충분 → S-N 선도 피로해석 별도(본 노트는 정적 위주, 피로 별도 노트 권장).

## 유한요소해석(FEM) 실무 (메시·경계조건·수렴)
- **경계조건은 실제 하중경로/구속을 반영.** 가짜 완전고정 면은 가짜 응력집중 유발 — 실제 체결/지지 방식 모사.
- **메시 수렴도 검토 필수**: 최소 2단계 세분화하며 최대응력·변위 추적, 변화율 2~5% 이내 안정 시 수렴 판단 [NAFEMS 기반].
- 전역 균일 대신 **응력집중부(필렛·홀·접촉·하중점)만 국부 세분화**.
- 체크리스트: 하중경로 타당 / 반력·모멘트 평형 / 구속이 인위적 강성 안 만듦 / 세분화·변수 변경에도 결과 안정.

## 자주 하는 검토 (볼트 체결부, 브래킷, 하우징)
- **볼트 체결부**: [[VDI 2230]] 또는 NASA-STD-5020B로 체결력(preload)·풀림·피로·나사산 전단 검증. 토크→축력 전환은 마찰계수에 따라 17~43% 편차, 조임계수(tightening factor) ~1.6 고려 [VDI 2230; NASA-STD-5020B].
- **브래킷/하우징**: 응력집중부(필렛 반경·개구부 모서리)가 파괴 시작점. 접촉조건 단순화(완전접합) 시 과소평가 주의.
- 방산 특성상 정적해석 외 발사/운송 충격·진동을 [[MIL-STD-810H]] Method 516/519로 별도 평가(정적 안전율 충족해도 동적 미검토 시 불충분).

## 관련 규격·소프트웨어
- 규격: NASA-STD-5020B([[나사·체결]]), [[VDI 2230]], FAR 25(안전율), [[MIL-STD-810H]](진동/충격).
- 소프트웨어: ANSYS Mechanical, [[solidworks]] Simulation(사용자 2022 기준).
- ⚠️ **사내/국방 규격(KS·국방규격·MIL 계열)이 있으면 항상 그것이 이 노트 참고치보다 우선.**

> 신뢰도 medium: NASA 안전율(1.5/1.4/1.25)은 1차 출처 확인이나 유도탄/국방 특화 규격은 미검색.
> 실제 프로젝트는 사내/국방 규격 최우선. 피로·좌굴 상세는 별도 노트 권장.
