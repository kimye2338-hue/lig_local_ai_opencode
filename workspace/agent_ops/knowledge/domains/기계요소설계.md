# 기계요소설계 기초

## ⚠️ 핵심 원칙 (항상)
- **정적 강도 + 피로(반복하중) 검토를 항상 함께.** 정하중 통과해도 반복하중서 피로파괴 가능(Shigley Ch.5-6).
- 안전율 n: 일반 기계요소 1.5~2.5, 인명·안전(방산 포함) 2.5~4↑ 통상. **구체값은 사내 설계기준/규격/고객요구 우선**("확인 필요").
- 응력집중 Kt → 피로 응력집중 Kf=1+q(Kt−1) (q=노치감도)를 축·기어 필렛에 반드시 반영.
- 표면처리(질화·침탄·쇼트피닝)는 피로한도 Se 개선하나 수치 편차 큼 — 공급사/실측 확인.
- 단위 혼용 주의: 기어 모듈(mm, ISO/KS) vs 다이아메트럴피치(1/in, AGMA). 규격 명시 없이 판단 금지.

## 축(Shaft) 설계 (비틀림·굽힘·피로·임계속도)
- **DE-Goodman 피로 설계(Shigley Eq.7-8, 표준)**:
  d = [ (16n/π)·( 2·Kf·Ma/Se + √3·Kfs·Tm/Sut ) ]^(1/3)
  (n 안전율, Kf/Kfs 굽힘/비틀림 피로집중, Ma 교번굽힘모멘트, Se 보정피로한도, Tm 평균비틀림, Sut 인장강도).
- **DE-ASME Elliptic(Eq.7-9, 항복 Sy 기준 병행, 다소 보수적)**: d=[(32n/π)·√((Kf·Ma/Se)²+(3/4)(Kfs·Tm/Sy)²)]^(1/3).
- Se = ka·kb·kc·kd·ke·kf·Se' (표면·크기·하중·온도·신뢰도·기타 보정 × 시편값). 각 계수 Shigley Table 6-x, 임의 대입 금지.
- 처짐/비틀림각: 기어축 처짐 ≤0.8mm/m, 비틀림 0.25~0.5°/m 권장(경험치, 사내기준 "확인 필요").
- 임계속도: 운전속도를 1차 고유진동수의 70~80% 이하(마진은 사내기준 확인). 키홈·단차·나사부는 Kt 도표(App.A-15) 필수.

## 기어(Gear) 설계 (모듈·치형·강도 AGMA/Lewis, 백래시)
- **Lewis 굽힘응력**: σ = Wt/(b·m·Y) (Wt 접선하중, b 치폭, m 모듈, Y Lewis 치형계수; 20°에서 Y≈0.154−0.912/Z, Z>12).
- **AGMA 굽힘(2001-D04 실무표준)**: σ = Wt·Ko·Kv·Ks·Km·KB/(b·mn·J) (Ko 과부하, Kv 동적, Ks 크기, Km 하중분포, KB 림, J 기하계수). Lewis에 실동하중 보정.
- **치면강도(pitting)**는 별도 AGMA 표면내구력(Hertz 접촉, 2101) 필요 — 굽힘만으로 종료 불가. 방산 동력전달 기어는 굽힘+치면 둘 다 필수.
- 모듈(mm, ISO/KS) vs DP=25.4/m(인치, AGMA) 확인 필수. 백래시 통상 0.03~0.05m(mm), 정밀도등급/용도 편차 큼("확인 필요").
- 헬리컬/베벨/웜은 추력·미끄럼 상이 — 스퍼 공식 직접대입 금지, 기어형별 AGMA 규격 참조.

## 베어링(Bearing) 선정 (수명 L10, 정/동정격하중)
- **기본 정격수명(ISO 281)**: L10 = (C/P)^p [백만 회전] (C 동정격하중, P 등가동적하중, p=볼 3·롤러 10/3).
- P = X·Fr + Y·Fa (X/Y는 형식·하중비별 계수 — 제조사 카탈로그 표 필수, 임의 대입 금지).
- 시간환산: L10h = L10×10^6/(60n) [시간]. 고신뢰도(방산)는 수정정격수명(SKF Lnm, 신뢰도·윤활·오염 aSKF) 권장.
- 정정격하중 C0(소성변형 한계, 저속·정지 하중)은 동정격 C와 별개 — 혼동 주의.
- 형식: 레이디얼→깊은홈볼/원통롤러, 큰 액시얼→앵귤러/테이퍼롤러, 자동조심→구면롤러. 방산 진동/충격은 예압·내식/고온 재질 별도.

## 스프링·키·커플링·체결요소
- **코일스프링(KS B 2400)**: τ = Kw·8FD/(πd³) (Kw Wahl 수정계수), k = Gd⁴/(8D³Na).
- **평행키(KS B 1311)**: 전단·압축(베어링) 두 조건 각각 검토해 큰 값 채택. 정확한 치수·계수는 KS B 1311 원문 표.
- **커플링**: 강성(오정렬 불허·고정밀) vs 플렉시블(오정렬 흡수·진동감쇠). 방산은 충격완충 필요 시 플렉시블 우선.
- **나사 체결**: 예압 Fi≈0.75·Fp(재사용, Shigley Ch.8), T=K·F·d (K 토크계수 ~0.2, 표면/윤활 편차 큼 — 실측 우선). **진동환경은 풀림방지(나일록·록타이트 등) 필수.**

## 실무 검토 ([[구조해석]] 연계)
- 축·기어·베어링·키는 동일 동력경로에서 연동 — 한 요소 강도 올리면 인접(축지름↑→베어링/키 재선정) 재확인.
- 손계산 근사식과 [[구조해석]] FEA 결과 상호검증. 괴리 시 FEA 우선하되 메시/경계조건 재확인.
- 최종 수치(안전율·표면계수·X/Y)는 재질증명서·제조사 카탈로그·사내 설계기준 실측/공인값 우선. 본 노트 경험치는 1차 스크리닝용.
- 제작은 [[기계공작법]]·[[CNC]], 체결 규격은 [[나사·체결]] 연계. ⚠️ 방산 국방규격이 있으면 그것이 우선.
