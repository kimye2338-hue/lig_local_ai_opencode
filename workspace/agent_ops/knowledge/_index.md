# 레퍼런스 지식베이스 — 전체 지도

방위산업 엔지니어 AI 조수의 큐레이션된 사실 지식. 폴더/링크로 무엇이 있는지 파악.
(개인 기억은 별개 — USERDATA/memory. 여기는 공식·출처 기반 레퍼런스.)

- (백로그) NASA-STD 계열 · VDI 2230(→[[기계요소설계]]§1.7)

## 소프트스킬 (lifeskills/)
- [[문서작성]] 기술문서·시험보고서·논문(IMRaD)·명확성 체크리스트
- ※ 협상/커리어/ADHD/성과관리는 레퍼런스 KB가 아닌 개인 기억/코칭 영역 → KB 제외(Fable 검토)

## 신뢰도 규약
각 노트 프론트매터의 `verified`(교차검증)·`confidence`(high/medium/low)·`sources`(출처) 확인.
수치·규격은 "출처에 이렇게 명시" 병기, 확실치 않으면 "확인 필요". **사내/국방 규격이 항상 우선.**
