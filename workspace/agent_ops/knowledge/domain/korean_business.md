# 한국 회사 업무 맥락 (메일·회의록·보고서 톤 가이드)

한국 회사(특히 방산/제조 대기업) 맥락에서 비서가 메일 초안·회의록·보고서·대외 커뮤니케이션을
만들 때 적용할 실무 감각. 아래는 널리 알려진 한국 비즈니스 관행을 우리 말로 재정리한 것
(특정 저작물 복제 아님, 원리 요약).

## 의사결정·결재 프로세스

- 의사결정은 **품의/결재 라인**을 탄다: 담당 → 검토 → 결재 문서 작성 → 결재 라인(팀장→부서장→…)
  → 예산 확정 → 계약. 서구보다 느리고(수 주~수개월) 여러 단계를 거친다.
- 문서는 **결재받기 좋게**: 목적/근거/기대효과/비용/일정이 한눈에. 결정권자가 빨리 판단하도록
  결론과 요청사항을 앞에 둔다.

## 언어·완곡 표현 해석

- **"검토해 보겠습니다 / 어렵습니다 / 고민해 보겠습니다"** 는 종종 **완곡한 거절/보류** 신호다.
  즉답 압박보다 여지를 남기고 후속 확인.
- 침묵/뜸들임은 거절이 아니라 **내부 논의 중**일 수 있다.
- 상급자·연장자에게는 존대와 격식. 직함(과장/차장/부장/이사)으로 호칭.

## 커뮤니케이션 매너

- 담당자를 건너뛰고 상급자에게 바로 접촉하지 않는다(라인 존중).
- 첫 미팅에서 일정·가격을 재촉하지 않는다. 관계·신뢰가 먼저.
- 사내 단체 채팅방/카카오톡은 간결·정중하게. 업무시간 외 연락은 신중히.
- 회식·경조사 등 관계 맥락을 존중("눈치" — 분위기와 상대 입장을 읽는 것).

## 문서 톤 (메일/회의록/보고서)

- **메일**: 인사 → 용건(두괄식) → 근거/첨부 → 요청/기한 → 맺음. 존대, 군더더기 없이.
- **회의록**: 일시/참석/안건 → 결정사항 → 액션아이템(담당·기한 명시) → 미결/이월.
- **보고서**: 결론 먼저, 근거·수치엔 출처/기준일. 과장 없이, 추정은 추정이라 표기.

## 적용 원칙

- 대외/상급자 대상 문서는 격식·존대·정중함을 기본값으로.
- 거절/부정 소식은 완곡하되 사실은 분명히. 애매하게 흐리지 않는다.
- 날짜·금액·담당은 구체적으로. 액션아이템엔 반드시 담당자와 기한.
