# 기계 설계 → 제조 → 가공 (무엇이 있나)

설계에서 실제 부품이 나오기까지의 흐름:

- **설계**: [[기계요소설계]] (축·기어·베어링·스프링·키·커플링·체결)
- **가공법 선정**: [[기계공작법]] (절삭·연삭·주조·단조·용접·열처리·표면처리·DFM·공차)
- **CNC 프로그래밍**: [[CNC]] (G/M코드·가공조건·사이클·검증)

## 연결
- [[기계요소설계]] 축/기어 → 제작은 [[기계공작법]]·[[CNC]], 강도검증은 [[구조해석]]
- [[기계공작법]] 공차(IT등급) ↔ [[CNC]] 가공조건 ↔ 설계 요구
- 진동·충격 요구는 [[진동해석]]·[[MIL-STD-810H]], 치구는 [[치구설계]](수집 예정)

> 한 부품 작업이 설계→공작법→CNC를 자연히 함께 끌어오도록 연결. ⚠️ 사내 공정표준·국방규격 우선.
