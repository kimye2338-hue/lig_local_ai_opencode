# 진동/충격/피로 — 무엇이 있나

- [[진동해석]] — SDOF~MDOF 모달·랜덤(PSD/Miles)·충격(SRS)·전달률·시험규격
- 피로수명: [[피로파괴]] (§5 랜덤진동 피로 — PSD→응력→레인플로우/Steinberg/Dirlik→Miner)
- 신호처리: [[데이터처리]] (FFT·PSD·Welch·레인플로우·계측)
- 관련 규격: [[MIL-STD-810H]] (Method 514 진동, 516 충격)
- 관련 도메인: [[구조해석]], [[치구설계]](진동시험 치구), [[재료역학]], [[유도탄설계]]
- 실행 소프트웨어: [[ANSYS]] Mechanical(모달/하모닉/랜덤), [[solidworks]] Simulation

## 대표 흐름
PSD 가진 → [[진동해석]] 응답(모달·Miles) → 응력 → [[피로파괴]] 수명 → [[MIL-STD-810H]] 시험 →
[[치구설계]] 지그(공진회피) → [[데이터처리]] 계측·분석

> 이 지도는 진동 관련 작업에서 '무엇을 참고할 수 있는지' 모델에게 먼저 보여준다.
> 하위주제가 늘면 여기 링크를 추가한다.
