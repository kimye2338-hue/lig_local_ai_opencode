# 진동해석 — 무엇이 있나

- [[진동해석]] — 기초(모달·랜덤·PSD·응답)
- 관련 규격: [[MIL-STD-810H]] (Method 514 진동, 516 충격)
- 관련 도메인: [[구조해석]], [[치구설계]], [[유도탄설계]]
- 실행 소프트웨어: [[ANSYS]] Mechanical, [[SolidWorks]] Simulation

> 이 지도는 진동 관련 작업에서 '무엇을 참고할 수 있는지' 모델에게 먼저 보여준다.
