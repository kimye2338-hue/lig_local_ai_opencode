## 정직한 결론 — 무엇이 남았나

이 환경은 오프라인이고 Windows cmd.exe·`opencode.exe`·실 LLM 게이트웨이·Obsidian 앱·Office/
SolidWorks가 **없습니다.** 그래서 아래는 "코드·설치기는 맞게 구현/수정됨"까지 확인했고 **실제
작동 스모크만 사내 PC 1회** 남았습니다. 이걸 "완료"로 단정하면 말씀하신 거짓말이 되므로 구분합니다:

1. **설치 스모크**: 설치기 `.bat` 실제 실행(경로·괄호·문법은 정적 검증 완료) → wheel 자동설치·ocd PATH 등록 확인
2. **ocd 실주행**: 새 CMD서 `cd 프로젝트폴더` → `ocd` → opencode 켜지고 폴더 페르소나 주입
3. **게이트웨이 실연결**: 첫 채팅이 EXAONE/Qwen에 실제 붙는지
4. **실 앱 자동화**: Excel/HWP/SolidWorks 실제 파일 생성(앱 설치 PC)
5. **야간 무중단 실동작**: 오케스트레이터를 켜두고 밤새 태스크 처리
