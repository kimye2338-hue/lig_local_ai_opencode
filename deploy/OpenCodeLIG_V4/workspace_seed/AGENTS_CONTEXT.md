# AGENTS_CONTEXT.md — 컨텍스트 사용 규칙 (필요할 때 읽는 참고 문서)

컨텍스트 한도는 약 32k 토큰으로 작다. 아껴 써야 한다.

## 읽어야 할 것 (필요할 때, 부분만)
- 현재 작업 대상 파일 — 가능하면 필요한 구간만.
- `checkpoints/CHECKPOINT_LATEST.md`, `memory/MEMORY.md` — 세션 시작 시.
- `skills/skill_*.md` — 해당 작업을 할 때만 1개씩.

## 읽지 말아야 할 것
- 로그 파일 전체 (마지막 20~40줄만 tail로).
- 대형 파일 전체를 한 번에 (부분 읽기 후 필요한 만큼 추가).
- 모든 스킬 파일을 한꺼번에.
- 이미 체크포인트/메모리에 요약된 과거 이력의 원본.

## 출력 절약
- 같은 내용을 반복 설명하지 않는다.
- 파일 본문 대신 경로+요약. 표보다 짧은 문장.
- 진단 결과는 [RESULT] 블록만 인용한다.
