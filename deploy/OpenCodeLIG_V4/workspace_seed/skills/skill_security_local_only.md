# skill_security_local_only — 사내망 전용 보안 규칙

## 핵심
이 PC는 폐쇄 사내망이다. 비밀은 로컬에만 존재해야 한다.

## 규칙
- 내부 API 키와 게이트웨이 URL은 `OpenCodeLIG\secrets\lig_local.env`에만 있다.
  - 이 파일을 read 도구로 열어 내용을 채팅에 출력하지 않는다.
  - 이 파일을 다른 파일로 복사하거나 코드에 하드코딩하지 않는다.
- git 커밋/푸시 대상에 secrets 폴더, *.env, 키 문자열이 포함되면 중단하고 경고한다.
- 공개 저장소에 올릴 파일에는 내부 호스트명(사내 도메인)도 쓰지 않는다.
- 설치/스크립트 작성 시 금지: PowerShell Bypass, Base64 ZIP/EXE 페이로드,
  난독화 코드, 외부 다운로드 (오프라인 원칙).
- 필요한 곳에서는 환경변수 이름만 언급한다: `LIG_API_KEY` (값 금지).
