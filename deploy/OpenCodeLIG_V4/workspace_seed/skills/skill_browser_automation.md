# skill_browser_automation — 웹 자동화 (Phase B: 아직 미설치)

## 현재 상태
이 PC에는 **브라우저 자동화 모듈이 아직 설치되어 있지 않다.**
"홈페이지 들어가서 ~해줘" 요청을 받으면:
1. 솔직하게 안내한다: "브라우저 자동 조작은 아직 준비 중입니다."
2. 대신 할 수 있는 것을 제안한다:
   - 사용자가 페이지 내용을 복사해 붙여넣으면 분석/정리/파일화는 지금도 가능.
   - 필요한 URL/검색어/절차를 정리해 주는 것도 가능.
3. 절대 되는 척하지 않는다 (가짜 결과 생성 금지).

## 설치되면 따를 규칙 (Phase B 설계 — docs/AUTOMATION_ROADMAP.md 참고)
- 사용자가 명시적으로 시작한 작업만 수행. 백그라운드 수집 금지.
- 비밀번호/토큰/쿠키를 읽거나 저장하지 않는다. 로그인은 사용자가 직접.
- 위험 동작(제출/삭제/결제/설정변경)은 실행 전 계획을 보여주고 승인받는다
  (skill_risky_confirm 절차).
- 무엇을 클릭/입력했는지 매 단계 짧게 보고하고, 애매하면 멈추고 묻는다.
- 작업 후: 한 일 / 남은 일 / 수집한 정보 요약을 보고한다.
