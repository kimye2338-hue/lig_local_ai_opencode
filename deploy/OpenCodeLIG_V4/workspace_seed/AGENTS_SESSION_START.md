# AGENTS_SESSION_START.md — 새 세션 시작 절차 (config instructions로 로드됨)

새 세션이 시작되면 무거운 탐색을 하기 전에 아래 순서를 따른다.

1. `checkpoints/CHECKPOINT_LATEST.md`가 있으면 read 도구로 읽는다.
   - 없으면: 새 작업으로 간주하고 3단계로 간다.
2. 체크포인트의 `미해결 이슈`와 `다음 행동`을 확인한다.
   - 사용자가 "계속" / "이어서" 라고 하면 `다음 행동`부터 재개한다.
3. `memory/MEMORY.md`를 읽는다 (반복 실수 방지 사실들이 있다).
4. 값비싼 진단(DIAG)은 사용자가 요청하거나 명백한 장애가 있을 때만 실행한다.
5. 작업 의도가 불명확하면 **질문은 1개만** 간결하게 한다. 명확하면 바로 실행한다.
6. 세션 종료 전 또는 큰 작업 완료 후: `SESSION_END_CHECKLIST.md`를 따라
   체크포인트를 갱신한다.
