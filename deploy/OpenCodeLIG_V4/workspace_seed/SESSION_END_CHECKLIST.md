# SESSION_END_CHECKLIST.md — 세션 종료 / 압축 직전 체크리스트

의미 있는 작업을 마쳤거나, 세션을 끝내거나, 컨텍스트 압축이 임박하면:

1. `checkpoints/CHECKPOINT_TEMPLATE.md` 양식으로
   `checkpoints/CHECKPOINT_LATEST.md`를 **write 도구로 덮어쓴다**.
   - 현재 작업 / 완료된 것 / 미해결 이슈 / 다음 행동(구체적 명령·파일) / 수정 파일 목록.
2. 새로 알게 된 지속 사실이 있으면 `memory/MEMORY.md`에 추가한다 (AGENTS_MEMORY.md 규칙).
3. 사용자에게 마지막으로 보고: 완료 요약 3줄 + 다음에 할 일 1줄.
   (파일 본문·로그 원문은 붙이지 않는다.)
